Cara Menggunakan KompresAja

  1. Klik tombol Unggah Gambar dan pilih file yang ingin Anda kompres.
  2. Setelah file terpilih, klik tombol Kompres untuk memulai proses kompresi.
  3. Tunggu proses selesai, lalu klik Unduh untuk menyimpan file hasil kompresi.
